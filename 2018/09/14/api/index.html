    参数 说明 备注 默认
    advertiser_id 广告主ID 必填
    advertiser_name 广告主名称 必填
    day_budget 每日预算 必填
    offers 活动列表 必填
    id 活动ID 必填
    name 活动名称 必填
    landing_page 落地页 必填
    country_code 国家定向 非必填
    term 终端定向 非必填 0:pc;1:移动 1
    plat 平台定向 非必填 0:iOs,1:Android,2:Window,3:Linux,4:Mac 全选
    region 地域定向 非必填(国家标准区划码 多地区用”,”分割)
    period 投放时段 非必填 week(1,2,3,4,5,6,7),hour(24位二进制,高位23点,低位0点) 全天
    status 活动状态 非必填 1:运行; 2:暂停 1
